### 🛠️ Tech Stack

Next.js 15 (App Router), TypeScript, Tailwind CSS, Lucide Icons

---

### ✨ Features

- Multi-tech AND / OR / NOT logic builder
- Typeahead tech picker (with tags)
- Filters:
  - Country, Industry, Minimum tech count, Technology category
- Pagination
- Open domain in new tab
- Clean, readable layout
- Export to CSV and JSON

---

### ⚙️ Setup & How to Run

Clone the project:

```bash
git clone https://github.com/RajK01/fiber-fullstack-challenge

cd fiber-fullstack-challenge
```
Install:
```bash
npm install
```
Setup for Datas
```bash
npm run setup
```
Run the Server

```bash
npm run dev
```
